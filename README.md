# vencord_background
Vencord simple background replacer / *recreated by me, original creator https://mwittrien.github.io/*

Allows to replace you background with vencord / betterdiscord any other discord mods.

Simply just put this css in your "Themes" Folder in vencord/bettercord

**If you want to make your own background go to 57 line of code and replace with any image from internet! / you need to copy the link to image.**
